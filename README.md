# 🎵 Flutter 音樂播放器

這是一款使用 Flutter 製作的簡單網頁版音樂播放器，具備基本的音樂播放功能，並支援狀態管理與等待動畫。

## 🔧 功能特色

- 🎼 支援本地音訊播放（使用 [audioplayers](https://pub.dev/packages/audioplayers) 套件）
- ✅ 簡易播放、暫停控制
- 🔄 音樂切換與狀態監控（使用 `Provider` 進行狀態管理）
- 🌐 支援 Flutter Web（部署至 GitHub Pages）
